# Assignment 2 – Divide and Conquer Algorithms

This repository contains the Python implementation and performance comparison of two divide-and-conquer algorithms:

- Merge Sort
- Quick Sort

The algorithms were tested on three datasets:
- Sorted data
- Reverse sorted data
- Random data

Performance metrics recorded:
- Execution time
- Memory usage

Files included:
- divide_and_conquer_comparison.py – main implementation
- execution_time_graph.png – performance comparison graph
- results.txt – experimental results
